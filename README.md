# Arduino-BlinkFor
Sketch Arduino che fa lampeggiare 3 led per diversi periodi

# Funzionamento del circuito

* Tasto 1: Blink led 3 volte
* Tasto 2: Blink led 5 volte
* Tasto 3: Blink led 8 volte

# Materiale utilizzato

* Scheda Arduino
* Breadboard
* Cavi Jumper
* 3 led
* 3 tasti
* 6 resistenze 220ohm
